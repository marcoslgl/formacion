# Nuxt-Vue-Exercises 🚀

Proyecto base utilizando **Nuxt 4**, **Composition API** y **TypeScript**.

## 🛠️ Stack Tecnológico

- **Framework:** Nuxt 4 (Vue 3)
- **Estilos:** Tailwind CSS
- **Estado:** Pinia
- **Tipado:** TypeScript

## 📁 Estructura Principal

- `.output/`, `.nuxt/`: Archivos generados por el framework.
- `assets/`: Recursos globales (imágenes, estilos SCSS).
- `components/`: Componentes reutilizables de Vue.
- `layouts/`: Plantillas maestras de la aplicación.
- `pages/`: Vistas de la aplicación (enrutamiento automático).
- `stores/`: Definiciones de Pinia para el estado global.
